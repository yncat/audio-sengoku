## Audio Sengoku python proto

ターンベースのやつです。

## Run

python 3がいります。

初回実行時には、

python3 -m pip install -r requirements.txt

などとして、必要なライブラリをインストールしてください。

次からは、

python3 boot.py

などとして起動できます。

## 操作方法
右キー (p1:右矢印 / p2: D): レーン移動右

左キー (p1:左矢印 / p2: A): レーン移動右

上キー (p1:上矢印 / p2: W): レーン移動右
